# Brainrot Hunt & Evolution

## Опис
Це прототип гри Roblox у 3-й особі з **ловлею brainrot-петов**, PvP, Rebirth та мультиплеєром на 6 гравців.  
Гравці знаходять хаотично спавнені pets у трьох компактних зонах і прокачують їх.

---

## 📌 Що всередині
- 3 зони спавну (`Zone1`, `Zone2`, `Zone3`)  
- Хаотичний спавн brainrotів з таймером **60 секунд**  
- Ловля та прокачка pets  
- PvP (гравці можуть красти pets один у одного)  
- Rebirth система із бонусами  
- Мультиплеєр на 6 гравців  
- UI для ловлі і прокачки  
- Камера від 3-ї особи  
- Стандартна спокійна музика Roblox  
- Папка `ReplicatedStorage/Pets` для твоїх brainrot-моделей

## 🗂️ Як запустити
1. Скачай `.rbxlx` файл з репозиторію.  
2. Відкрий Roblox Studio → **File → Open → Brainrot_Hunt_Prototype.rbxlx**.  
3. Додай свої brainrot-моделі у `ReplicatedStorage → Pets` та вкажи атрибут **Rarity** (`Common`, `Rare`, `Epic`, `Legendary`, `Secret`).  
4. Перевір гру через **Test → Players → Start**, встановивши до 6 гравців.

---

## ⚡ Як додавати моделі pets
- Кожен brainrot повинен мати унікальну модель у `ReplicatedStorage → Pets`.  
- Вкажи атрибут **Rarity** для правильного спавну та балансу.  

---

## 🎵 Музика
- Стандартна спокійна музика Roblox вже додана через `SoundService`.  

---

## 📝 Примітки
- Таймер спавну всіх brainrotів **60 секунд** незалежно від рідкості.  
- Зони компактні для активного геймплею.  
- PvP активне тільки в межах зон спавну.  
- Rebirth скидає pets і очки, але дає постійні бонуси (шанс рідкісних pets, x2 BrainPoints).
